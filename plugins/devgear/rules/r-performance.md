# パフォーマンス標準

## メモ化・キャッシュ

- 高コストな計算は結果を再利用
- 依存値が変わらない限り再計算しない
- キャッシュ無効化条件を明確に

## 遅延読み込み

- 重要でないリソースは後で読む
- 初期表示を重くしない
- 読み込み中の状態を適切に示す

## DBクエリ

- 必要なカラムだけ取得
- LIMITやインデックスを活用
- N+1問題を避ける
